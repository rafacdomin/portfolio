## Colaboration Guide

### Commit patterns

|  Emoji   |  Name  | Use case                           |
| :------: | :----: | :--------------------------------- |
| :scroll: | scroll | Ex.: :scroll: update documentation |
| :pencil: | pencil | Ex.: :pencil: configure prettier   |
|  :gear:  |  gear  | Ex.: :gear: configure node version |
| :broom:  | broom  | Ex.: :broom: clear project         |
